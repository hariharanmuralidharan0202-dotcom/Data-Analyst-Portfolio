# Netflix Movies & TV Shows EDA #

## Project Overview
This project performs Exploratory Data Analysis (EDA) on Netflix dataset to understand content trends and provide business insights.

##  Objective
- Analyze Movies vs TV Shows
- Identify trends over the years
- Understand country-wise content distribution
- Provide recommendations for Netflix

##  Dataset
- Source: Kaggle Netflix Dataset
- Contains information about movies and TV shows including genre, cast, country, and release year.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Movies dominate the platform but TV Shows are growing rapidly
- Content production increased after 2015
- USA is the leading content producer

## Business Recommendations
- Invest more in TV Shows
- Focus on regional content
- Expand in growing markets like India & South Korea
